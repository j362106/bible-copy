온전히 집중

오직 말씀과 내 손글씨만. 주의를 끄는 다른 것 없이 조용히 필사에만 집중합니다.

🎵

잔잔한 음악

필사하는 동안 마음을 가라앉혀 주는 배경 음악을 켤 수 있습니다.

💾

자동 저장 · 비공개

필사 기록은 자동으로 저장되고 내 컴퓨터를 벗어나지 않습니다. 로그인도, 클라우드도 없습니다.

🛡️ 처음 실행하시나요?

작은 무료 프로그램이라, 실행하면 Windows가 파란색 “Windows의 PC 보호” 화면을 띄울 수 있습니다. 안전합니다 — 오픈소스, 퍼블릭 도메인 앱입니다. 이렇게 하세요:

  1. 추가 정보 클릭
  2. 실행 클릭
